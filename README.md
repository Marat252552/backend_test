ЗАПУСК

Инициализация проекта и установка пакетов с yarn

В .env файл добавить следующие переменные:
PORT =

    MY_SQL_HOST =
    MY_SQL_USER =
    MY_SQL_PASSWORD =
    MY_SQL_DATABASE =

    JWT_ACCESS_KEY =
    JWT_REFRESH_KEY =

Запуск (yarn dev)

Для внесения изменений запустите typescript компилятор (yarn tsc -w)
